## linhas de comandos:

PRINCIPAL: cat filename.sql | sqlite3 movies.db


1.sql - Escreva uma consulta SQL para listar os títulos de todos os filmes lançados em 2008.

    Comando: sqlite3 movies.db "SELECT title FROM movies WHERE year = 2008;" > 1.sql



2.sql - Escreva uma consulta SQL para determinar o ano de nascimento de Emma Stone.

    Comando: sqlite3 movies.db "SELECT birth FROM people WHERE name = 'Emma Stone';" > 2.sql



3.sql - Escreva uma consulta SQL para listar os títulos de todos os filmes com data de lançamento igual ou posterior a 2018, em ordem alfabética.

    Comando: sqlite3 movies.db "SELECT title FROM movies WHERE year >= '2018' ORDER BY title;" > 3.sql



4.sql - Escreva uma consulta SQL para determinar o número de filmes com uma classificação IMDb de 10,0.

    Comando 1: sqlite3 movies.db "SELECT COUNT(title) FROM movies WHERE id IN (SELECT movie_id FROM ratings WHERE rating = '10.0');" > 4.sql
    comando 2: sqlite3 movies.db "SELECT COUNT(title) FROM movies JOIN ratings ON ratings.movie_id = movies.id WHERE rating = '10.0';" > 4.sql



5.sql - Escreva uma consulta SQL para listar os títulos e anos de lançamento de todos os filmes de Harry Potter, em ordem cronológica.

    Comando: sqlite3 movies.db "SELECT title, year FROM movies WHERE title LIKE '%Harry Potter%' ORDER BY year;" > 5.sql



6.sql - Escreva uma consulta SQL para determinar a avaliação média de todos os filmes lançados em 2012.

    Comando 1: sqlite3 movies.db "SELECT AVG(rating) FROM ratings WHERE movie_id IN (SELECT id FROM movies WHERE year = '2012');" > 6.sql
    Comando 2: sqlite3 movies.db "SELECT AVG(ratings.rating) FROM ratings JOIN movies ON movies.id = ratings.movie_id WHERE year = '2012';" > 6.sql



7.sql - Escreva uma consulta SQL para listar todos os filmes lançados em 2010 e suas classificações, em ordem decrescente por classificação. Para filmes com a mesma classificação, ordene-os em ordem alfabética por título.

    Comando: sqlite3 movies.db "SELECT movies.title, ratings.rating FROM movies JOIN ratings ON ratings.movie_id = movies.id WHERE year = "2010" ORDER BY rating DESC, title ASC;" > 7.sql



8.sql - Escreva uma consulta SQL para listar os nomes de todas as pessoas que estrelaram Toy Story.

    Comando: sqlite3 movies.db "SELECT name FROM people JOIN stars ON people.id = stars.person_id JOIN movies ON stars.movie_id = movies.id WHERE title = 'Toy Story';" > 8.sql



9.sql - Escreva uma consulta SQL para listar os nomes de todas as pessoas que estrelaram um filme lançado em 2004, ordenado por ano de nascimento.

    Comando: sqlite3 movies.db "SELECT DISTINCT(name) FROM people JOIN stars ON people.id = stars.person_id JOIN movies ON stars.movie_id = movies.id WHERE year = '2004' ORDER BY birth;" > 9.sql



10.sql - Escreva uma consulta SQL para listar os nomes de todas as pessoas que dirigiram um filme que recebeu uma classificação de pelo menos 9,0.

    Comando: sqlite3 movies.db "SELECT DISTINCT(name) FROM people JOIN directors ON people.id = directors.person_id JOIN movies ON directors.movie_id = movies.id JOIN ratings ON movies.id = ratings.movie_id WHERE rating >= 9.0;" > 10.sql



11.sql - Escreva uma consulta SQL para listar os títulos dos cinco filmes com melhor classificação (em ordem) que Chadwick Boseman estrelou, começando com os de maior classificação.

    Comando: sqlite3 movies.db "SELECT title FROM movies JOIN stars ON movies.id = stars.movie_id JOIN people ON people.id = stars.person_id JOIN ratings ON ratings.movie_id = movies.id WHERE name = 'Chadwick Boseman' ORDER BY rating DESC LIMIT 5;" > 11.sql


12.sql - Escreva uma consulta SQL para listar os títulos de todos os filmes em que Johnny Depp e Helena Bonham Carter estrelaram juntos.

    Comando: sqlite3 movies.db "SELECT title FROM movies JOIN stars ON movies.id = stars.movie_id JOIN people ON stars.person_id = people.id WHERE people.name = 'Johnny Depp' OR people.name = 'Helena Bonham Carter' GROUP BY title HAVING COUNT(title) > 1;" > 12.sql



13.sql - Escreva uma consulta SQL para listar os nomes de todas as pessoas que estrelaram um filme no qual Kevin Bacon também estrelou.

    Comando: sqlite3 movies.db "SELECT DISTINCT(name) FROM people JOIN stars ON people.id = stars.person_id JOIN movies ON movies.id = stars.movie_id WHERE title IN (SELECT DISTINCT(title) FROM people JOIN stars ON people.id = stars.person_id JOIN movies ON stars.movie_id = movies.id WHERE name = 'Kevin Bacon' AND birth = 1958) AND name != 'Kevin Bacon';" > 13.sql
